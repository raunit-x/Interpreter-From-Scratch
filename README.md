<h3> Type of Parser </h3>
<h5>Recursive Descent Parser with Backtracking</h5>
<h3> Features </h3>
<ul>
  <h4>1. Math Operations (Add, subtract, multiply, divide, exponentiation) with both int and float types</h4>
  <img src='images/Screenshot 2020-07-22 at 1.34.19 PM.png' width='600' />
  <h4>2. Support for variable assignments of int, float, string, list and even functions</h4>
  <img src='images/Screenshot 2020-07-22 at 1.39.19 PM.png' width='600' />
  <h4> 3. Support for string operations </h4>
  <ul>
    <li>Supports string concatenation</li>
    <li>Supports repeating the string</li>
  </ul>
   <img src='images/Screenshot 2020-07-22 at 1.43.05 PM.png' width='600' />
  <h4> 4. Support for List operations and the list is generic and supports nesting! </h4>
  <ul>
    <li>Supports negative indexing</li>
    <li>Supports appending the list with an element</li>
    <li>Supports extending the list with another list</li>
    <li>Supports popping the list with valid indexes</li>
  </ul>
   <img src='images/Screenshot 2020-07-22 at 1.48.17 PM.png' width='600' />
  <h5> 5. Support for Conditional statements </h5>
  <ul>
    <li>Supports IF, ELIF and ELSE</li>
    <li>Nesting upto any level</li>
    <li>Can even assign variables the expr of the if-else ladder like a terenary operation</li>
  </ul>
   <img src='images/Screenshot 2020-07-22 at 1.57.01 PM.png' width='600' />
  <h5> 5. Support for Loops </h5>
  <ul>
    <li>FOR and WHILE loops are supported</li>
    <li>Can set the step of the iterable in the FOR loop</li>
    <li> Can store the result of the LOOPS in a list </li>
  </ul>
   <img src='images/Screenshot 2020-07-22 at 2.06.09 PM.png' width='600' />
</ul>
